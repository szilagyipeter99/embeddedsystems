This example demonstrates the usage of the manufacturer provided embedded driver for the Sensirion SHT40, SHT41 and SHT45 humidity and temperature sensors.

Original repo here:
https://github.com/Sensirion/embedded-i2c-sht4x/tree/master

As stated by Sensirion, the sensirion_i2c_hal.c file should be edited depending on the hardware platform.
